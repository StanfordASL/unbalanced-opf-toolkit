Unbalanced OPF Toolkit's documentation
***************************************
A general code base to formulate and solve unbalanced optimal power flow (OPF) problems written in *MATLAB*. 
|uot| formulates the optimization problems using `YALMIP <https://yalmip.github.io/>`_ :cite:`Lofberg2004` allowing to use a multitude of solvers. 
|uot| supports `GridLAB-D <https://www.gridlabd.org/>`_ :cite:`Chassin2014` models of the power distribution network.


Citing Unbalanced OPF Toolkit
===============================
If you use Unbalanced OPF Toolkit for a scientific publication, please cite :cite:`Estandia2019`.


.. note ::

   The documentation is a work in progress. 
   If you would benefit from documentation in a specific set of files please create an issue so that they get higher priority.
